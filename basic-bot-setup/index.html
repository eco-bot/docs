    Basic Bot Setup

    Setting up the prefix

    I want the bot to respond when I use the prefix - to do that, I can type eb!set prefix -.


    Setting up all the channels

    eb!set role-logs <#channel> Sets the channel where Ecobot will log things such as role create, role delete, & role update. More on that later.

    eb!set welcome-channel <#channel> Decides where join messages go, more on that later.


    Setting up modlogs

    We can use the eb!set modlogs <#channel> command set the modlogs to the channel you have mentioned. Whenever a user is warn/kick/softbaned/baned the bot would send a message there


    Settings Join/Leave Logs

    First we will set the message sent to the previously set channel with:

    • eb!set welcome-channel <#channel> posts a random message when a user joins the server.
    • eb!set leave-channel <#channel> posts a message when a user leaves the server.
    • eb!set join-role <@role> add the role the newly joining member.
    Note: Make sure bot has is placed above the role you want to give
