    How do I read this guide?

    Danger

    Do not literally type out < > [ ] || etc

    Each category of commands has their own page which can be found on the sidebar.

    Aliases: [foo||bar] means that you can use either foo or bar.
    Optional: [foo] means that this argument can be ignored.
    Required: means that you must use this argument for the command to work.
    Many: or [foos...] means that you can specify more than one. If you wish you use an argument with more than one word, use "double quotes" to let the bot you know what you want.
